Analisis Transportasi Besar
Dashboard analisis data untuk memantau performa rute dan tren penumpang transportasi.

Deskripsi Proyek
Proyek ini dibangun untuk mendemonstrasikan kemampuan dalam:
Data Generation: Membuat dataset simulasi besar (500.000+ baris) menggunakan Python.
Data Analysis: Melakukan eksplorasi data menggunakan Pandas (Groupby, Aggregation).
Visualization: Membangun dashboard interaktif menggunakan Streamlit dan Plotly.

Fitur Dashboard
1. Analisis Rute: Visualisasi total penumpang per rute.
2. Analisis Cuaca: Distribusi jumlah penumpang berdasarkan kondisi cuaca.
3. Statistik: Ringkasan statistik deskriptif data.

Teknologi yang Digunakan
Python: Bahasa pemrograman utama.
Streamlit: Framework untuk aplikasi web dashboard.
Plotly: Library untuk visualisasi data interaktif.
Pandas: Library untuk manipulasi data.

Cara Menjalankan
1. Clone repository ini.
2. Install dependencies: `pip install -r requirements.txt` (jika ada).
3. Jalankan aplikasi: `streamlit run app.py`
